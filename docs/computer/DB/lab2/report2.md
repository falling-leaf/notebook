1. 掌握关系数据库语言SQL的使用。
2. 使所有的SQL作业都能上机通过。

1. 建立数据库。

4. 数据查询：单表查询、多表查询、嵌套子查询等。
5. 视图操作：通过视图的数据查询和数据修改。
6. 所有的SQL作业都能上机通过。
